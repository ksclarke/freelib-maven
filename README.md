This project is a small collection of Maven plugins. It requires &gt;= JDK 17.

## Getting Started

### How to Build

To build the project, type:

    git clone https://github.com/ksclarke/freelib-maven-plugins.git
    cd freelib-maven-plugins
    mvn install
